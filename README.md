### 关于范围

### 关于性别和种族
用v-if控制，选择完敏感属性之后就会消失
用的是范围输入框
对于种族(race_White)：[0, 0]:非白人, [0, 1]:全选, [1, 1]:白人
对于性别(sex_Male)：[0, 0]:女, [0, 1]:全选, [1, 1]:男
**必须先提交表单，要不然后面的无法进行**
